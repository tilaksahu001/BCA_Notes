Unit 2: Introduction to C++ and OOP

1. Object-Oriented Programming (OOP)

OOP is based on the concept of “objects”, which bundle data and functions. It uses principles like:

  • Encapsulation: Data hiding within classes.
  • Abstraction: Show only necessary details.
  • Inheritance: One class derives from another.
  • Polymorphism: Multiple behaviors with the same interface.

2. Features of C++

  • Supports procedural and OOP.
  • High performance and control over memory.
  • Cross-platform source code compatibility.

3. Structure of a C++ Program

#include <iostream>

using namespace std; int main() { cout << "Hello, World!"; return 0; }

4. Data Types in C++

  • Primitive: int, float, char, bool
  • Derived: array, pointer
  • User-defined: class, struct, union

5. Class and Object

class Student {

public: int id; string name; void display() { cout << "ID: " << id << ", Name: " << name; } }; Student s1; s1.id = 101; s1.name = "Tilak"; s1.display();

6. Constructors and Destructors

Special functions to initialize and clean up objects.

  • Default Constructor: No arguments
  • Parameterized Constructor: Accepts parameters
  • Copy Constructor: Copies from another object

7. Access Specifiers

  • Public: Accessible outside the class
  • Private: Accessible only inside the class
  • Protected: Accessible in inherited classes

8. Inline Functions

inline int square(int x) {

return x * x; }

9. Static Members

Shared among all objects. Used for tracking object count, etc.

10. Summary

This unit introduces core OOP features in C++ like encapsulation, inheritance, polymorphism, and memory control, laying the foundation for scalable software development.
