Programming in C - Unit 1

1. What is Programming?

Programming is the process of writing instructions that a computer can understand and execute. These instructions are written in programming languages like C, C++, Python, etc.

2. What is a Programming Language?

A programming language is a formal language with rules, syntax, and keywords used to communicate with computers.

  • Low-Level Languages: Assembly, Machine Language
  • High-Level Languages: C, C++, Java, Python
  • 4GL: SQL, MATLAB

3. Programming Paradigms

  • Procedural
  • Object-Oriented
  • Functional

4. Example of a Simple C Program

#include <stdio.h>

int main() { printf("Hello, World!"); return 0; }

5. Example of a Simple C++ Program

#include <iostream>

using namespace std; int main() { cout << "Hello, World!"; return 0; }

6. Definition of a Program

A program is a set of instructions written in a language that tells the computer what to do.

7. Compilation and Execution

  • Compilation: Converts source code to machine code.
  • Execution: Runs the compiled machine code.

8. Algorithm Example

Algorithm to Add Two Numbers:

  1. Start
  2. Read A and B
  3. Sum = A + B
  4. Print Sum
  5. Stop

9. C Implementation

#include <stdio.h>

int main() { int A, B, Sum; scanf("%d %d", &A, &B); Sum = A + B; printf("Sum = %d", Sum); return 0; }
